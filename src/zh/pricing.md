## 开源自托管 (Self-Hosted)
如果您拥有自己的基础设施并乐于自托管，您依然可以免费使用易连（EasyConnect SSH）：
- **核心功能**: 100% 免费开源
- **部署方式**: 遵循 **[服务端指南](/zh/guide/server-usage)** 自行部署
- **技术支持**: 通过 GitHub Discussions 获取社区支持
