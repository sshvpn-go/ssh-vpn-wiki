# 我们的使命与隐私承诺

SSHVPN 拥抱精简到极致的可用性构建哲学。为了打造一个不会向中间平台妥协的绝对安全的连线隧道，开源社区的爱好者们共同设计了这一切。

## 拒绝企业商业监视

传统的消费级 VPN 服务要求用户完全信赖厂商的数据审计行为，且总是会在连线过程中收集不可查证的网络统计指标。

**而 SSHVPN 彻底粉碎了这种监控架构。**
由于每一次的数据转发全都在开源的 SSH 建立和验证层面实现，我们将复杂的虚拟网络变成了“直连服务”。

- **拥有 100% 的节点归属权:** SSHVPN 会与完全归您所有的代理或海外服务器形成“点对点加密”，不经过厂商黑盒路由审核。
- **天然对抗包头嗅探:** 商业监控平台极其容易锁定并绞杀普通 VPN 的连接协议，而 SSH 作为基石协议，无论对审查者或运营商来说都是无差别网络数据的存在。
