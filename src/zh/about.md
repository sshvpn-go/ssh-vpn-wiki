# 我们的使命与隐私承诺

易连助手（EasyConnect SSH，代码 `ssh-vpn`）的设计目标很明确：提供一个可审计、基于 SSH 的远程接入客户端，服务于企业团队在合法前提下进行内网访问、远程运维与自管基础设施连接。

## 明确的控制权与可审计性

许多远程接入产品依赖托管式网关或不透明的客户端组件。易连助手（EasyConnect SSH）将模型保持在尽可能简单的范围内：直接连接到由您或所在组织控制的 SSH 服务器。

- **接入端点由您掌控：** 易连助手（EasyConnect SSH）直接与您或所在组织指定的 SSH 服务器通信。
- **开源且可审计：** 项目代码公开，便于团队审查流量转发与配置管理的实现方式。
- **强调合法用途：** 文档面向远程管理、企业内网访问、开发测试环境接入等合规场景。
- **面向全球合规：** 软件在不同国家和地区使用时，应同时满足当地法律法规、行业要求、数据跨境规则以及所在组织政策。

软件的部署与使用应始终遵守使用地的适用法律法规、数据与网络管理要求，以及所在组织的安全与合规制度。
