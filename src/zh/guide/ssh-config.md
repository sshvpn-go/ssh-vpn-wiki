# 客户端配置文件解析 (`profile.json`)

尽管绝大部分人只需依靠图形界面 (GUI) 愉快地开关节点，但底层那份格式精巧的 JSON 配置文件，才是高级开发者用来定制脚本自动运维的终局入口。

您的节点设置都会以该格式在本地被安全隔离储存，下面我们一起解构 `profile.json` 所提供的高级自定义参数（适用于版本号 `1.0` 以上）。

---

### 一、连接校验与身份特征
构建第一层不可透传加密握手的必备要素：

| JSON 字段 | 数据格式 | 中文释义 |
|---|---|---|
| `name` | 字符串 | 用户自由定案的节点别名（如：东京-测试节点01）。 |
| `host` | 字符串 | 远端 SSH 服务器的公网 IP 或是被解析的可通达域名。|
| `port` | 整数 | SSH 隧道目标端口号（默认为标准 `22` 端口）。 |
| `username` | 字符串 | 服务器上能够认证合法的账号名称。 |
| `auth_type` | 字符串 | 控制免密登陆通道，只允许 `"password"`（密码校验） 和 `"key"`（私钥校验）。 |
| `password` | 字符串 | 纯明文或混淆储存于底层的连接密码（私钥模式时请直接留空）。 |
| `key_path` | 字符串 | 如果使用私钥校验，务必填入对应的绝对磁盘路径如 `~/.ssh/id_rsa`。 |

### 二、网络接口与执行环境
这决定了您是要全局劫持所有应用的流量通道还是仅仅搭建一个安静的本地代理。

| JSON 字段 | 数据格式 | 中文释义 |
|---|---|---|
| `mode` | 字符串 | `"socks5"` 只提供端口级的本地代理转发，没有提权弹窗； `"tun"` 激活最高全网拦截并重设物理路由路径的底层接管。 |
| `socks5_port`| 整数 | 指定仅能在 `"socks5"` 模式下生效的本机 SOCKS 服务器对外端口（默认为 `1080`）。 |
| `tun_name` | 字符串 | 如果您想客制化网卡的名字可以指定（例如 `sshvpn0`），否则留空会自动选择（如 Mac 下的 `utun0`）。 |
| `tun_ipv4` | 字符串 | 发配给该底层局域虚拟网卡的子网配置值（默认推荐使用 `10.0.0.1/24`）。 |
| `tun_ipv6` | 字符串 | 同理，配发给该网卡的 IPv6 子网掩码地址。 |
| `enable_system_proxy` | 布尔值 | 当处于 SOCKS5 模式下，允许程序主动调取操作系统 API，帮您省去手动进入控制面板帮 Chrome 或 Safari 设置代理的麻烦。 |

### 三、智能分流与抗 DNS 中毒屏蔽 (重要！)
这部分用于掌控您的数据到底应该走光缆直球到达国内机房，还是潜行过海绕过多国检测。

| JSON 字段 | 数据格式 | 中文释义 |
|---|---|---|
| `enable_rules` | 布尔值 | 主控开关：决定接下来的分流和抗屏蔽解析引擎到底要不要参与处理这批数据包。 |
| `rules_path` | 字符串 | 本地磁盘中指向 `.txt` （如主流的 `china_ip_rules`）规则集文件的确切地址。 |
| `bypass_local` | 布尔值 | 启用后彻底将诸如 `192.168.0.0/16` 这种局域网通讯自动放行，以节省贵如黄金的海外跳板机流量开销。 |
| `enable_local_dns` | 布尔值 | （仅限 TUN 模式下有效）将本地 `53` 端口绑定一个安全的防抖 DNS ，彻底斩断运营商故意投送污染结果导致上不去境外网的可能。 |
| `dns_servers` | 队列 | 一旦上面功能开启，系统应该相信并询问哪些安全域（如 `["8.8.8.8", "1.1.1.1"]`）。 |

### 四、应对复杂网络状况的参数调优
为了防止诸如校园网被挤占丢包，或是移动高铁信号频繁切换断联的微操场景。

| JSON 字段 | 数据格式 | 中文释义 |
|---|---|---|
| `auto_reconnect` | 布尔值 | 面对极端信号变动隧道瞬间死亡时，允许服务进程死缠烂打无限次重链并补包。 |
| `connect_timeout`| 字符串 | 初始化那一次 TCP 握手的忍耐时间到底需要几秒钟（默认推荐 `"10s"`，如果您发现很难连上也可以修改为 `"30s"` 等候）。 |
| `keep_alive_interval` | 字符串 | 每隔多久发射一个心跳微波，用来蒙蔽那些喜欢随意处死连接空闲端口的流氓防火墙机器（默认推荐 `"30s"`）。 |
