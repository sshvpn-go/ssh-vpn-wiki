## ADDED Requirements

### Requirement: Easy Link Assist Documentation
The documentation site MUST include comprehensive user guides and reference pages for the Easy Link Assist product, detailing screen sharing, remote control features, settings, and security guidelines.

#### Scenario: Accessing Easy Link Assist Guide in English
- **WHEN** the user navigates to `/assist/guide`
- **THEN** they see the English user manual covering Intro, Features, Quick Start, and Security warnings for Easy Link Assist.

#### Scenario: Accessing Easy Link Assist Guide in Chinese
- **WHEN** the user navigates to `/zh/assist/guide`
- **THEN** they see the Chinese user manual (易连友助使用指南) with the same features and security guidelines.
